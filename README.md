# 💼 Lost Wallet Finder

A simple web application to help people report and find lost wallets.

## Features

- **Report Lost Wallet**: Users can post details about their lost wallet including location, date, and description
- **Report Found Wallet**: Users can report wallets they've found to help return them to owners
- **Search Database**: Browse all reported lost and found wallets with filters
- **Local Storage**: All wallet data is stored in the browser's local storage (no backend required)
- **Contact Information**: Direct email and phone contact details for wallet owners/finders
- **Responsive Design**: Works on desktop and mobile devices

## How to Use

1. **Report Lost Wallet**: Fill in the form with your wallet details and contact information
2. **Report Found Wallet**: If you found a wallet, post it so the owner can find it
3. **Search**: Browse the database and search by location, color, or other details

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Local Storage API

## Getting Started

1. Clone this repository
   ```bash
   git clone https://github.com/ustinmabini-rgb/btc.git
   ```

2. Open `index.html` in your web browser

3. Start reporting and searching for wallets!

## Notes

- Data is stored locally in your browser (clears if you clear browser data)
- For production use, consider adding a backend database
- No sensitive financial data should be stored
